<h3>Infinite, structured memory for long-running autonomous agents.</h3>

<i>Replacing flat, easily confused vector databases with dynamic knowledge graphs, this SKILL.md extension dictates how the agent extracts entities and relationships from conversations. It builds a local graph allowing the agent to remember complex facts accurately over years of use.</i>
